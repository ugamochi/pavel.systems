FAQ

Common implementation questions

Can this handle messy PDF formats?

Yes, with schema-specific extraction prompts and confidence fallback rules.

What happens on low confidence?

Records are quarantined to reviewer flow, not written into production systems.

Can we start with one document family?

Yes. Starter scope is intentionally narrow for fast value and low rollout risk.

Can we add AI later?

Yes. Rule-first extraction/validation can run before AI-assisted enrichment.
